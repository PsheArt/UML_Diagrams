## Описание класса Client 
Класс для работы с клиентами тату-салон

### Атрибуты 
+ ID:Int
+ FIO:String
+ Phone:String
+ Email:String 


### Описание атрибутов
+ ID:Int - идентификатор в БД
+ FIO:String - ФИО клиента
+ Phone:String - номер телефона клиента 
+ Email:String  - электронная почта клиента
